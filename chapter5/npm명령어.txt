// 앞 npm은 생략해서 작성됨
outdated : 어떤 패키지에 기능 변화가 생겼는지 알 수 있음
uninstall 패키지명 : 패키지 삭제(npm rm 패키지명 으로도 가능)
search 검색어 : npm 패키지를 검색할 수 있음(npmjs.com에서도 가능)
info 패키지명 : 패키지의 세부 정보 파악 가능
adduser : npm에 로그인하기 위한 명령어(npmjs.com에서 회원가입)
whoami : 현재 사용자가 누구인지 알려줌
logout : 로그인한 계정을 로그아웃
version : pakage.json의 버전을 올림
deprecate [패키지명][버전][메시지] : 패키지를 설치할 때 경고 메시지를 띄우게 함(오류가 있는 패키지에 적용)
publish : 자신이 만든 패키지를 배포
unpublish --force: 자신이 만든 패키지 배포 중단(배포 후 72시간 이내에만 가능)
ls : 내 프로젝트가 어떤 패키지를 사용하고 있는지 찾고 싶을 때 사용
기타 명령어는 https://docs.npmjs.com의 CLI Commands에서 확인
